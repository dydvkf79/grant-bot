# 🤖 AI 콘텐츠 지원사업 자동 브리핑 봇

매일 아침 **09:30 (한국시간)**에 한국·글로벌 AI 콘텐츠 지원사업/공모를
자동으로 훑어 **이메일로** 정리해 보내주는 봇입니다.
서버 필요 없음 · 완전 무료 (GitHub Actions 사용).

---

## 📦 구성 파일
- `collect.py` — 수집 엔진 (한국 공식 API + 기관 게시판 + 글로벌 RSS)
- `.github/workflows/daily.yml` — 매일 09:30 자동 실행 설정
- `requirements.txt` — 필요 라이브러리

---

## 🚀 셋업 (딱 4단계, 약 20분)

### 1단계 — GitHub에 올리기
1. github.com 가입/로그인 → 오른쪽 위 **+** → **New repository**
2. 이름 아무거나 (예: `grant-bot`), **Private** 선택 → Create
3. 생성된 페이지의 **"uploading an existing file"** 클릭
4. 이 폴더의 파일을 **그대로 드래그**해서 올리고 Commit
   (`.github` 폴더 구조 유지 중요)

### 2단계 — 한국 공식 API 키 발급 (무료)
1. **data.go.kr** 접속 → 회원가입
2. 검색창에 **"한국콘텐츠진흥원 지원사업공고"** 검색
3. 해당 OpenAPI에서 **활용신청** (자동 승인)
4. 마이페이지 → 인증키에서 **일반 인증키(Decoding)** 복사

### 3단계 — Gmail 발송 설정 (앱 비밀번호)
> 일반 비밀번호 말고 "앱 비밀번호"가 필요합니다.
1. Google 계정 → 보안 → **2단계 인증** 켜기 (필수)
2. 검색창에 "앱 비밀번호" → 새 앱 비밀번호 생성 → 16자리 복사

### 4단계 — 비밀값(Secrets) 입력
GitHub 저장소 → **Settings → Secrets and variables → Actions →
New repository secret** 에서 아래를 하나씩 추가:

| 이름 | 값 |
|------|-----|
| `DATA_GO_KR_KEY` | 2단계에서 복사한 인증키 |
| `SMTP_HOST` | `smtp.gmail.com` |
| `SMTP_PORT` | `587` |
| `SMTP_USER` | 본인 Gmail 주소 |
| `SMTP_PASS` | 3단계 앱 비밀번호(16자리) |
| `MAIL_TO` | 받을 주소 (여러 개면 쉼표로, 비우면 본인에게) |

## 🔧 자주 손볼 곳 (collect.py 상단)
- **`KEYWORDS`** — 관심 키워드 추가/삭제 (예: "환경", "음악" 등)
- **`EXCLUDE`** — 제외할 노이즈 키워드
- 글로벌 소스 추가는 `fetch_global_rss()` 의 `feeds` 리스트에 URL 추가

## ⚠️ 알아둘 점
- **사이트 구조가 바뀌면** 해당 스크래퍼(`fetch_나라`)가 빈 결과를 낼 수
  있습니다. 그래도 봇은 멈추지 않고 나머지 소스로 메일을 보냅니다.
  깨진 소스는 메일에 ⚠️ 표시로 나타나니 그때 그 함수만 고치면 됩니다.
- GitHub Actions 무료 한도(월 2,000분) 안에서 하루 1회 실행은 한 달에
  약 5분 수준이라 **사실상 평생 무료**입니다.
- 한국 공식 API가 가장 안정적이고, 게시판 스크래핑은 보조입니다.
